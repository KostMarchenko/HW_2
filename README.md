# HW 2: OOP, Testing, Git
Проект реализует систему для работы с рецептами. В программе есть классы для ингредиентов, обычных и диетических рецептов, а также списка покупок, который объединяет ингредиенты из разных рецептов.

## Установка
Склонируйте репозиторий:
```bash
git clone https://github.com/KostMarchenko/HW_2
cd HW_2
```
Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование
Код программы находится в файле `recipes.py`.
Запуск кода:
```bash
python recipes.py
```
Тесты находятся в файле `test_recipes.py`.
Запуск тестов:
```bash
pytest
```

## Использованные конструкции Python
В проекте используются генераторные выражения и `lambda`-функции. Генераторные выражения применяются для компактного перебора ингредиентов, а `lambda` используется как ключ сортировки в методе `sort`. Эти конструкции являются частью стандартного синтаксиса Python и описаны в официальной документации Python:
- https://docs.python.org/3/tutorial/datastructures.html
- https://docs.python.org/3/howto/functional.html
